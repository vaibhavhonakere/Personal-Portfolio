# Vaibhav Honakere Portfolio

Recruiter-focused personal portfolio built with React and Vite.

## Scripts

```bash
npm install
npm run dev
npm run build
```

## Structure

- `src/App.jsx` - page composition
- `src/components/*` - section components
- `src/data/siteContent.js` - portfolio content and links
- `src/styles.css` - visual system and responsive layout
